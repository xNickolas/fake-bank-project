# FakeBank

> fake Bank é um projeto desenvolvido pelo Grupo Angulado!


## Pré-requisitos

Para utilizar esse projeto você precisa de:

| Plugin | Versão |
| ------ | ------ |
| Node.js | v14.15.5 |
| Npm | v6.14.11 |
| Angular cli | v11 |
| Typescrit | v4.2 |

## Instalação e rodando projeto localmente

> Dentro da página fake-bank

```sh
npm install
ng serve 
```
